\section{System Components and Architecture}

Smart-Fall Detection System is built on a multi-layer IoT architecture that connects everything efficiently. 
It's designed such that data flows easily from the IoT device to the cloud application, and then sends back an alert right away. 
Using communication methods that save battery power, this creates a reliable and practical system that watches over the patient all the time.

        \hline
        \textbf{Layer} & \textbf{Components} & \textbf{Description} \\
        \hline
        Sensing & MPU6050 & Collects accelerometer and gyroscope raw data. \\
        \hline
        Edge Processing & ESP32 microcontroller & Collects 100 samples from the MPU6050 sensor and computes statistical features (mean, max, min, variance) for all six variables (\textbf{Ax, Ay, Az, Gx, Gy, Gz}). \\
        \hline
        Communication & Wi-Fi, MQTT protocol & MQTT and Wi-Fi are used for \textbf{power-efficient} publish–subscribe communication between the IoT node and the backend server. \\
        \hline
        Analytics & Random Forest Model & A pretrained Random Forest model with 24 features is deployed on the backend server to identify fall patterns. \\
        \hline
        Actuation & LED, Buzzer & Actuates the fall alert signal. \\
        \hline
        Application & React Frontend Dashboard & Provides real-time monitoring, prediction analytics, and a dashboard for data labeling and model retraining. \\
        \hline
