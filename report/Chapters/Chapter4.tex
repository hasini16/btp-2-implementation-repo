\lhead{Experimental Setup}

\section{Dataset Preparation}

\subsection{Custom Data Collection}
Simulated CNC sensor data (vibration/temp from literature/historical + synthetic via MPU6050 emulation). ~5000 points labeled (healthy/faulty/spindle wear).

Used cnc\_automation\_project backend DB for storage.

\subsection{ML Model Selection}
Trained RF, XGBoost, LSTM, CNN on features (mean/var/FFT). RF: 95\% acc; LSTM RUL RMSE 5\%. Best: hybrid RF-LSTM.

\subsection{Hardware Setup}
Sensors on CNC: vib (MPU), temp/ current. ESP32 processes → MQTT to backend/api/ml\_routes.py (final\_cnn\_model.keras).

Power: 5V adapter. Actuation: Alert via dashboard/email.
