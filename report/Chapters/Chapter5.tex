\section{Challenges Faced}
\begin{enumerate}
    \item Data scarcity: Limited real CNC failure data; used simulation + lit datasets.
    \item Model selection: Small data favored RF over DL; addressed via augmentation.
    \item Real-time edge processing: Feature extraction latency on ESP32 optimized.
    \item CAD integration: API heterogeneity; prototyped STEP-NC mapper (future).
    \item WiFi/battery in shop floor; used MQTT QoS2.
    \item XAI for PdM trust \cite{wakhare2023optimizing}.
\end{enumerate}
