\abstract{\addtocontents{toc}{} % Add a gap in the Contents, for aesthetics

This B.Tech Project develops an AI-driven Predictive Maintenance (PdM) system for CNC machines to minimize unscheduled downtime, a major cost in manufacturing. The system collects real-time multi-sensor data (vibration, temperature, acoustics) from CNC setups, extracts features on edge devices (e.g., ESP32), and employs cloud-based ML models (Random Forest, LSTM/CNN) for fault prediction and Remaining Useful Life (RUL) estimation. Key features include data collection mode for labeling, real-time prediction dashboard (React frontend), model retraining, and integration considerations for CAD/CAM workflows via STEP-NC for Industry 4.0. Built on the cnc\_automation\_project backend, it addresses literature gaps in lightweight, interpretable PdM with IoT.

\textbf{\textit{Keywords - }} CNC Predictive Maintenance, Machine Learning, LSTM, Random Forest, Industry 4.0, IoT, CAD/CAM, STEP-NC
}
